# SMIF: Social Media Intelligence Filter-inator

*Still under development.

This is a Python implementation of Facebook Search Tool from IntelTechniques and Sowsearch. Users can search the following types:
 - Posts
 - Photos
 - Videos
 - People
 - Events
 - User (Profile) Info
 - Generic Search

## Additional Developments
This is still being polished. The tool will also support other social media, such as Twitter (X) and Instagram.
